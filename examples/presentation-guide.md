# Nevron Presentation Guide

Rules and templates for creating on-brand PowerPoint/Keynote presentations.

---

## Setup

- **Dimensions:** 16:9 (1920 x 1080 px)
- **Margins:** 80px from all edges (safe zone for content)
- **Font:** Neue Haas Unica (or Inter as fallback)
- **Logo placement:** Bottom-right corner, within safe zone

---

## Slide Types

### 1. Title Slide

The opening slide. Sets the tone for the entire presentation.

```
┌──────────────────────────────────────────┐
│                                          │
│  ██████████████████████████████████████  │  Background: M1 Navy (#000126)
│                                          │  or gradient M1 → M3
│                                          │
│         PRESENTATION TITLE               │  White, Display (48px), Bold
│         Subtitle or date                 │  M5 Light Blue, Body Large (18px)
│                                          │
│                                          │
│                                          │
│                          [Logo+tagline]  │  White logo variant, bottom-right
└──────────────────────────────────────────┘
```

**Rules:**
- Title: White, 48px, Bold (700)
- Subtitle: M5 `#C3E9FF`, 18px, Regular (400)
- Use logo **with tagline** (white variant)
- Background: Solid M1 or gradient from M1 to M3

### 2. Section Divider

Marks the start of a new section. Keep it bold and simple.

```
┌──────────────────────────────────────────┐
│                                          │
│  ██████████████████████████████████████  │  Background: M3 Brand Blue (#1B92FF)
│                                          │
│  01                                      │  M5 Light Blue, Caption (12px)
│                                          │
│  Section Title                           │  White, H1 (36px), Bold
│                                          │
│                                          │
│                                          │
│                          [Logo no-tag]   │  White logo, no tagline
└──────────────────────────────────────────┘
```

**Rules:**
- Section number: M5 `#C3E9FF`, 12px, caps, wide letter-spacing
- Title: White, 36px, Bold (700)
- Background: Solid M3

### 3. Content Slide

The workhorse. Used for text, bullet points, and explanations.

```
┌──────────────────────────────────────────┐
│                                          │
│  Slide Title                             │  M1 Navy, H2 (28px), SemiBold
│  ─────────────────────                   │  M3 accent line (optional)
│                                          │
│  • Bullet point one                      │  S1 Near Black, Body (16px)
│  • Bullet point two                      │
│  • Bullet point three                    │  Background: White
│                                          │
│                                          │
│                          [Logo no-tag]   │  Blue logo, no tagline
└──────────────────────────────────────────┘
```

**Rules:**
- Title: M1 `#000126`, 28px, SemiBold (600)
- Body text: S1 `#080C13`, 16px, Regular (400)
- Bullet color: M3 Brand Blue
- Max 6-8 lines of text
- One key idea per slide
- Background: White

### 4. Image + Text (Split)

For showcasing product screenshots, photos, or illustrations alongside text.

```
┌──────────────────────────────────────────┐
│                                          │
│  ┌─────────────┐  Title                  │  50/50 or 60/40 split
│  │             │                         │
│  │   IMAGE     │  Description text       │  Image: full-height, rounded corners
│  │             │  goes here with         │  or full-bleed
│  │             │  supporting details.    │
│  │             │                         │
│  └─────────────┘  [CTA or link]          │
│                                          │
│                          [Logo no-tag]   │
└──────────────────────────────────────────┘
```

**Rules:**
- Image side: High quality, never stretched
- Text side: Background White or M6 Pale Blue
- Title: M1, H3 (22px), SemiBold
- Body: S1, Body (16px)
- Optional CTA: M3 blue link

### 5. Data / Chart Slide

For statistics, graphs, and metrics.

```
┌──────────────────────────────────────────┐
│                                          │
│  Slide Title                             │  M1 Navy, H2 (28px)
│                                          │
│  ┌──────────────────────────────────┐    │
│  │                                  │    │  Chart area
│  │        [CHART / GRAPH]           │    │  Colors: M3, M4, M5, S3
│  │                                  │    │  Labels: S2, Body Small (14px)
│  └──────────────────────────────────┘    │
│                                          │
│  Source: description               Logo  │
└──────────────────────────────────────────┘
```

**Rules:**
- Chart colors (in order of priority): M3 `#1B92FF`, M4 `#73C7FF`, M5 `#C3E9FF`, S3 `#707379`
- Labels and axes: S2 `#353941`, 14px
- Source text: S3 `#707379`, 12px
- Keep charts simple — no 3D effects, no excessive gridlines
- Data labels should be readable at presentation distance

### 6. Closing Slide

Final slide. Leaves a lasting impression and provides next steps.

```
┌──────────────────────────────────────────┐
│                                          │
│  ██████████████████████████████████████  │  Background: M1 Navy
│                                          │
│         Thank You                        │  White, H1 (36px), Bold
│                                          │
│         name@nevron.eu                   │  M5, Body (16px)
│         nevron.eu                        │  M4, Body (16px)
│                                          │
│                                          │
│                          [Logo+tagline]  │  White logo with tagline
└──────────────────────────────────────────┘
```

**Rules:**
- Match title slide aesthetic
- Include contact info and/or CTA
- Logo **with tagline** (white variant)

## General Rules

| Rule | Details |
|------|---------|
| **Text limit** | Max 6-8 lines per slide |
| **One idea** | One key message per slide |
| **Images** | Never stretch. Use high-res. Crop intentionally |
| **Emphasis** | Use M3 blue for emphasis — not red, not bold |
| **Animations** | Subtle only. Fade and slide. No bouncing or spinning |
| **Charts** | Use brand blues (M3-M5). No 3D effects |
| **Consistency** | Same margins, font sizes, and positioning across all slides |
| **Contrast** | Ensure text is readable on its background |
